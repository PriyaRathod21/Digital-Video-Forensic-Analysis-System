# Digital-Video-Forensic-Analysis-System
A Python-based forensic analysis system that detects video tampering, duplicate/near-duplicate frames, frame deletions, and compression artifacts, and generates a detailed forensic authenticity report.

Features:

* Extracts video metadata

* Captures all frames

* Detects exact & perceptual duplicate frames

* Detects suspicious frame deletions

* Analyzes compression artifacts

* Generates JSON forensic report

* Saves visualization of duplicate frames

* Prints formatted forensic summary
